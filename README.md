# Hava Durumu Uygulaması

Bu proje, WeatherAPI kullanarak şehirlerin anlık hava durumunu gösteren bir web uygulamasıdır.

## Kurulum

1. Repo’yu klonla: `git clone https://github.com/fatma-gokcen/Hava-durumu.git`
2. Klasöre gir: `cd Hava-durumu`
3. Paketleri yükle: `npm install`
4. Sunucuyu başlat: `node server.js`
5. Tarayıcıdan aç: `http://localhost:3000`

## Teknolojiler
- HTML, CSS, JS
- Node.js, Express
- WeatherAPI
